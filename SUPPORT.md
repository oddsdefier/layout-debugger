# Support

## Before asking for help

- Read the installation and usage instructions in [README.md](README.md).
- Reload the extension from `chrome://extensions/` after changing files.
- Refresh the page being inspected after changing content-script files.
- Confirm that the page is not a restricted browser URL.
- Search existing issues for the same problem.

## Bugs

Open a bug report using the repository’s bug template. Include the browser version, operating system, steps to reproduce, expected result, actual result, and screenshots or a short recording when useful.

## Feature ideas

Open a feature request and explain the debugging problem the feature would solve. Concrete examples are more useful than a proposed implementation alone.

## Security issues

Do not open a public issue. Follow [SECURITY.md](SECURITY.md).

## Scope

Support is provided on a best-effort basis. The project currently targets Chromium-based browsers with Side Panel support. Requests for unrelated browsers may be considered, but compatibility is not guaranteed.
