# Roadmap

Layout Debugger is early-stage but usable. This roadmap is directional and does not guarantee delivery dates.

## Near term

- Add screenshots or a short demo to the documentation.
- Improve keyboard navigation and accessibility in the side panel.
- Add repeatable manual test cases for overlays, selection, and DOM navigation.
- Improve error handling for unsupported pages and disconnected tabs.
- Review requested browser permissions and keep them as narrow as practical.

## Later

- Show more useful Flexbox and Grid diagnostics.
- Add tools for identifying overflow, clipping, stacking, and spacing problems.
- Allow users to copy or export a compact layout report.
- Investigate compatibility with additional Chromium-based browsers.
- Evaluate Firefox support without weakening the Chromium experience.

## Contributor-friendly work

Issues suitable for a small first contribution should be labeled `good first issue`. Larger work should begin with an issue describing the problem and expected behavior.

Ideas and corrections are welcome through the feature request template.
